# ChargerOS Project Info

ChargerOS is an Ubuntu-inspired browser desktop environment built by Flamechargerr.

## Stack

- Node.js 20+
- React 19
- TypeScript 5.9
- Vite 7
- Tailwind CSS 3.4
- Radix/shadcn-style UI primitives
- Lucide React icons

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Public Positioning

Use: "Ubuntu-inspired browser desktop OS simulation."

Avoid: "bootable Ubuntu OS" or "Linux distribution" until an ISO/build pipeline exists.
