# Resume Bullets

## Best One-Line Summary

Built **ChargerOS**, an Ubuntu-inspired browser desktop environment using React, TypeScript, Vite, and Tailwind CSS, featuring a 59-app suite, draggable window manager, app launcher, terminal, virtual filesystem, and local persistence.

## Strong Resume Bullets

- Built a browser-based Linux desktop simulation with React, TypeScript, Vite, and Tailwind CSS, implementing a full desktop shell, app launcher, top panel, taskbar, wallpaper, and desktop icons.
- Implemented a custom window manager supporting draggable/resizable windows, minimize/maximize/restore, close actions, z-index focus handling, and running-app indicators.
- Designed a 59-app modular suite across System, Internet, Office, Graphics, Media, Development, Games, and Utilities categories.
- Created a virtual Unix-style filesystem persisted through browser localStorage, including file read/write/delete, folder creation, rename, move, and `/etc/os-release` metadata.
- Built a Linux-style terminal with filesystem-aware commands including `ls`, `cd`, `pwd`, `mkdir`, `touch`, `cat`, `rm`, `echo`, `uname`, `neofetch`, `tree`, and `history`.
- Added portfolio-grade UI flows including login, guest entry, desktop context menu, app search, category navigation, persistent wallpaper, and app window state.
